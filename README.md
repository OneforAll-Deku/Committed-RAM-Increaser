# Committed RAM Increaser

A lightweight, automated tool designed to prevent "Out of Memory" errors in high-demanding games and heavy applications by optimizing and increasing your Windows Virtual Memory (PageFile) limit.

- **Disables** "Automatically manage paging file size" to prevent Windows from restricting your virtual memory.
- **Sets a Custom PageFile Size** on your C: drive ensuring you have enough committed memory.
- **Intelligent Calculation**: Automatically suggests an optimal size (typically 16GB - 32GB) based on your system RAM.
- **One-Click Operation**: Simple menu-driven interface.
