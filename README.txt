Committed RAM Increaser
=======================

This tool is designed to prevent "Out of Memory" errors in games and heavy applications by increasing your Windows Virtual Memory (PageFile) limit.

How to use:
1. Right-click "RamIncreaser.bat" and select "Run as Administrator" (or just double-click it, it will ask for permission).
2. Press '1' (or just Enter) when prompted to apply the recommended settings.
3. Restart your computer.

What it does:
- Disables "Automatically manage paging file size".
- Sets a custom PageFile size on your C: drive.
- Recommended Size: Starts at 16GB, up to 32GB (or higher for systems with lots of RAM).

To Uninstall/Revert:
- Run the tool again and select Option [2] "Revert to Windows Default".
